# Handlebars for Visual Studio Code

## Features
* Colorization
* Snippets

## Development

This project uses [Bun](https://bun.sh) as the package manager.

### Prerequisites
- [Bun](https://bun.sh) >= 1.0.0

### Setup
```bash
# Install dependencies (if any are added in the future)
bun install
```

Note: This is a simple language extension that requires no compilation or build steps.
